# Implementation Plan: Upgrade xUnit 3 and Dependencies

**Input**: Feature specification from `/specs/00012-upgrade-xunit3-deps/spec.md`

## Summary

Upgrade all NuGet packages in the Messaggero solution to their latest stable releases. The primary change is migrating the three test projects (Unit, Contract, Integration) from `xunit` 2.9.3 to `xunit.v3` 3.2.2 — a package rename migration that also requires adding `<OutputType>Exe</OutputType>` to each test project. Secondary changes are straightforward version bumps for all remaining stale dependencies across source and test projects. No C# source code modifications are required.
